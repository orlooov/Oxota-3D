# 🌲 Зов Природы (Oxota-3D)

**Учебный 3D-симулятор охоты от первого лица с элементами выживания и квестовой системой, разработанный на Unity.** 

Проект создавался в рамках курсовой работы для изучения механик 3D-игр, программирования искусственного интеллекта (NPC/Враги) и проектирования архитектуры игровых систем.

## 🎮 О проекте
Игрок берет на себя роль охотника, прибывшего в суровую таежную местность. Местные волчьи стаи начали вести себя аномально агрессивно. Взаимодействуя со старым охотником Егором (NPC), игроку предстоит провести расследование, выполнить цепочку заданий и выследить опасного Альфа-волка.

## 🛠️ Стек технологий
* **Движок:** Unity 3D
* **Язык:** C#
* **Моделирование:** Blender
* **Паттерны и инструменты:** State Machine (ИИ), Scriptable Objects (Квесты/Диалоги), NavMesh, Raycast.

## ⚙️ Реализованные механики (Что внутри кода)

В этом проекте я с нуля написал следующие системы:

* **🐺 Искусственный интеллект животных (Animal AI):**
  Написан скрипт поведения волка на базе `NavMeshAgent` с использованием паттерна состояний (State Machine). Волк умеет: патрулировать территорию, замечать игрока, преследовать и атаковать.
* **🔫 Система оружия (Weapon System):**
  Реализована логика ружья (Benelli M4). Стрельба реализована через `Raycast`. Присутствует система перезарядки, учет патронов в магазине и инвентаре, отдача и визуальные эффекты.
* **📜 Менеджер квестов (Quest Manager):**
  Глобальный менеджер, отслеживающий прогресс. Квесты имеют различные условия: таймеры, условия скрытности (Стелс) и требования к типу оружия. 
* **💬 Диалоговая система:**
  Возможность взаимодействия с NPC (Егор) для получения заданий и продвижения по сюжету.

## 💡 Чему я научился на этом проекте
1. Работать с 3D-пространством и ландшафтом (Terrain) в Unity.
2. Программировать ИИ с помощью NavMesh.
3. Разделять логику (Игрок, Оружие, Менеджеры квестов) для чистоты кода.
4. Работать со звуком (AudioSource) и анимациями (Animator).
